# v0.3.1-internal.2

## Scope

Internal testnet patch cut from the current `dev` head. This release includes
the shared market/subgraph fix required for the live testnet surfaces and the
guarded Robomata/Sui stabilization work accumulated after `v0.3.0-internal.1`.

## Included Work

- Moves `/markets` latest-earnings reads to subgraph-backed
  `EarningsDistribution` entities instead of browser RPC event scans.
- Adds deployed-network subgraph ABI selection so generated subgraph ABIs match
  deployed contracts.
- Keeps public-testnet browser RPC usage limited to the remaining wallet and
  contract-read flows.
- Carries forward Robomata submission API gating, Supabase-compatible Postgres
  storage, Sui sponsor signing hardening, and operator facility auto-assignment.

## Deployment Notes

- Redeploy hosted subgraphs for every active public testnet environment:
  - `sepolia`
  - `polygon-amoy`
  - `arbitrum-sepolia`
  - `monad-testnet`
- Revalidate `/markets` after each subgraph redeploy by confirming the page no
  longer emits public-provider `eth_getLogs` rate-limit errors for earnings
  history.
- Keep the release tagged separately from the Robomata product tag so the
  internal testnet line can be audited independently.
